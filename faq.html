สามารถดาวน์โหลดเอกสารประกอบการเรียนได้ที่เมนู "ดาวน์โหลด" หรือที่ หน้าดาวน์โหลดเอกสาร โดยเอกสารจะแบ่งเป็นหมวดหมู่ตามบทเรียน

  1. ตรวจสอบเนื้อหาที่เรียนจากสไลด์การสอนในหน้าดาวน์โหลด
  2. ดูวิดีโอบันทึกการสอน (หากมี)
  3. ติดต่อเพื่อนร่วมชั้นเรียนเพื่อขอสรุปเนื้อหา
  4. หากมีข้อสงสัยสามารถติดต่อผู้สอนผ่านช่องทางในหน้า ติดต่อผู้สอน

สำหรับรายวิชานี้แนะนำให้ติดตั้งโปรแกรมต่อไปนี้:

  • MySQL Community Server - ระบบจัดการฐานข้อมูล
  • MySQL Workbench - เครื่องมือจัดการฐานข้อมูล MySQL
  • SQLite - ระบบฐานข้อมูลแบบไฟล์
  • Text Editor - เช่น Visual Studio Code, Sublime Text

สามารถดาวน์โหลดโปรแกรมเหล่านี้ได้ที่หน้า ซอฟต์แวร์และเครื่องมือ

หากมีปัญหาในการติดตั้ง MySQL ให้ทำตามขั้นตอนต่อไปนี้:

  1. ตรวจสอบว่าคอมพิวเตอร์ของคุณตรงกับระบบที่ต้องการหรือไม่
  2. ดาวน์โหลดไฟล์ติดตั้งใหม่จากเว็บไซต์ทางการ
  3. ปิดโปรแกรมแอนตี้ไวรัสชั่วคราวขณะติดตั้ง
  4. ติดตั้งด้วยสิทธิ์ Administrator
  5. ตรวจสอบคู่มือการติดตั้งในหน้า ดาวน์โหลด
  6. หากยังมีปัญหา สามารถส่งรายละเอียดข้อผิดพลาดมาที่ ผู้สอน

การส่งงานหลังจากกำหนดเวลาจะมีการหักคะแนนดังนี้:

  • ส่งช้าไม่เกิน 1 สัปดาห์: หัก 20% ของคะแนนเต็ม
  • ส่งช้าเกิน 1 สัปดาห์: หัก 50% ของคะแนนเต็ม
  • ส่งช้าเกิน 2 สัปดาห์: ไม่รับการส่งงาน

ข้อยกเว้น: หากมีเหตุจำเป็นต้องแจ้งผู้สอนล่วงหน้า

สามารถแก้ไขงานที่ส่งแล้วได้เฉพาะก่อนถึงกำหนดส่งเท่านั้น โดยทำตามขั้นตอนต่อไปนี้:

  1. เข้าไปที่หน้า ส่งงาน
  2. เลือกงานที่ต้องการแก้ไข
  3. กดปุ่ม "แก้ไขงาน"
  4. อัปโหลดไฟล์ใหม่และยืนยันการส่ง

หมายเหตุ: หลังจากหมดเวลาส่งแล้วไม่สามารถแก้ไขงานได้

สอบกลางภาค: เนื้อหาตั้งแต่บทที่ 1-5 ประกอบด้วย

  • แนวคิดระบบฐานข้อมูล
  • การออกแบบฐานข้อมูลเชิงสัมพันธ์
  • คำสั่ง SQL พื้นฐาน
  • การสร้างและจัดการตาราง
  • การสืบค้นข้อมูลเบื้องต้น

สอบปลายภาค: เนื้อหาตั้งแต่บทที่ 6-10 และทบทวนบางส่วนจากกลางภาค

สามารถตรวจสอบตารางสอบและเนื้อหาอย่างละเอียดได้ที่ ปฏิทินเรียน
